# Setup Guide

## Prerequisites
- Docker
- Python 3.9+
- Terraform

## Installation
1. Clone the repository: `git clone [repository-url]`.
2. Install dependencies: `pip install -r requirements.txt`.
3. Configure the system: Edit `config.yml`.

## Deployment
1. Build Docker images: `docker-compose build`.
2. Start the system: `docker-compose up`.

For detailed instructions, refer to the [Installation Guide](docs/setup/installation.md) and [Deployment Guide](docs/setup/deployment.md).
